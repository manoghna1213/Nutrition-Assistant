# 🍎 Nutrition Assistant

## 📌 Project Overview
Nutrition Assistant is a full‑stack wellness management platform built using the MERN stack. It enables users and dietitians to create, manage, monitor, and analyze personalized nutrition plans. The system is designed to be scalable, secure, and responsive, following clean architecture principles.

## ⚙️ Features
- Personalized nutrition plan creation and tracking  
- Role‑based access for users, dietitians, and admins  
- Analytics dashboard for monitoring progress  
- Secure authentication with JWT  
- Responsive UI with React and Material UI
